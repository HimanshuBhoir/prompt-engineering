## Summary: Domain-Specific Prompting

**Coding:** Specify language, provide codebase context, request tests/docs, use scaffolding scripts, follow style guides

**Teaching:** Define learner level, use Socratic method, scaffold content, provide actionable feedback, personalize to learner

**Research:** Structure information extraction, synthesize multiple sources, verify claims, identify gaps, maintain academic rigor

**Product:** Provide business context, use frameworks (RICE, SWOT), customize for stakeholders, balance user/business/technical needs

**Writing:** Define voice/style, specify structure, match audience, iterate and refine, optimize for purpose (SEO, engagement, conversion)

**The meta-pattern:** Domain expertise + clear context + structured output + examples = effective domain-specific prompts.
